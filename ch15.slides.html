from matplotlib.colors import ListedColormap
import numpy as np

def plot_decision_boundary(X, y, classifier, resolution=0.02):
    markers = ('o', 'x', 's') # 
    colors = ('red', 'green', 'blue') # 
    cmap = ListedColormap(colors[:len(np.unique(y))])

    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution),
                     np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha = 0.3, cmap = cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())
    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x = X[y == cl, 0], y = X[y == cl, 1], c = cmap(idx), marker = markers[idx], label = cl)

決策樹¶

$$IG(D_p) = I(D_p) - \frac{N_{left}}{N_p}I(D_{left}) - \frac{N_{right}}{N_p}I(D_{right})$$
    • Error
    • Gini Index
    • Entropy*

決策樹（2）¶

  • Error
  • Confusion Matrix
5 5
6 6
  • $\text{Accuracy}= \frac{5 + 6}{5 + 5 + 6 + 6} = 0.5$
  • $\text{Error} = 1 - \text{Accuracy} = 1 - 0.5 = 0.5$

決策樹（3）¶
$1 - accuracy$
$1 - accuracy^2 - error^2$
$-(accuracy\log_2(accuracy) + error\log_2(error))$

決策樹（4）¶

  • A
$I(D_p)$ $I(D_{left})$ $I(D_{right})$
$40/80 = 0.5$ $10/40 = 0.25$ $10/40 = 0.25$
$1-0.5^2-0.5^2 = 0.5$ $1-0.75^2-0.25^2 = 0.375$ $1-0.75^2-0.25^2 = 0.375$
$-(0.5log_2(0.5)+0.5+0.5log_2(0.5)+0.5) = 1$ $-(0.75log_2(0.75)+0.25log_2(0.25)) = 0.81$ $-(0.75log_2(0.75)+0.25log_2(0.25)) = 0.81$

決策樹（5）¶

  • B
$I(D_p)$ $I(D_{left})$ $I(D_{right})$
$40/80 = 0.5$ $20/60 = 1/3$ $0/20 = 0$
$1-0.5^2-0.5^2 = 0.5$ $1-(2/3)^2-(1/3)^2 = 1/3$ $1-1^2-0^2 = 0$
$-(0.5log_2(0.5)+0.5+0.5log_2(0.5)+0.5) = 1$ $-(2/3log_2(2/3)+(1/3)log_2(1/3)) = 0.92$ $-(1log_2(1)+0log_2(0)) = 0$

決策樹（6）¶

A B

$0.5-\frac{4}{8}\times0.375-\frac{4}{8}\times0.375=0.125$ $1-\frac{6}{8}\times0.\bar{4}-0=0.1\bar{6}$
$0.5-\frac{4}{8}\times0.81-\frac{4}{8}\times0.81=0.125$ $1-\frac{6}{8}\times0.92-0=0.31$
